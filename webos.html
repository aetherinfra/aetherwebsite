ARMv8 AArch64 Architecture

Designed specifically for the next generation of ARM servers. Aether WebOS features direct hardware interaction, including:

  • • Exception Level (EL) management
  • • GICv2/v3 Interrupt handling
  • • Stage 1 MMU translation
  • • Generic Timer initialization

Headless UART Support

Precision debugging and administration via high-speed UART serial interfaces. Built for data centers where local display output isn't an option.

  • • Real-time kernel log streaming
  • • Interactive PL011 UART shell
  • • Zero-overhead debug prints
